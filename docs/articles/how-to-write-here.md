# How to Write Here

This is a starter article. Duplicate this file when you want to create a new post, rename it with a simple URL-friendly name, and update the frontmatter at the top.

## Frontmatter

Every article should start with this block:

```md
---
title: Your Article Title
description: A one-sentence summary for previews and search engines.
date: 2026-05-09
---
```

## Writing

Write the rest of the article in normal Markdown:

- Use `#` for the page title.
- Use `##` for major sections.
- Use links, lists, quotes, and code blocks whenever they help.

When the article is ready, add it to the article list in `docs/articles/index.md`.
